    About

    VOCALOID Heardle is a daily song guessing game built around VOCALOID and vocal synth music.

    Each puzzle reveals a little more of the intro after every wrong guess or skip. Song details and VocaDB links appear after the answer.

    Support

    VOCALOID Heardle is maintained by sodapines as a small fan-made music puzzle project.

    For bugs, song corrections, or suggestions, email kzen@sodapines.dev.

    Hosting and the stats backend cost a bit each month to keep running. If you've enjoyed the game and want to chip in, even a one-time tip helps cover those costs.

    Release Notes

    Release v1.6

    This release collects the major updates being prepared for the public site.

    • Added Unlimited song filters for producer, voicebank, program, publish year, and community-suggested songs.
    • Filtered Unlimited now has separate local stats, so filtered streaks no longer mix with normal Unlimited stats.
    • Active filters appear as removable tags in the top tag row.
    • Filters can be inclusive, letting players build broader pools from multiple criteria.
    • Changing filters during an unfinished Unlimited round counts that round as a loss.
